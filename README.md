🎵 LyricFinder - Real-Time Music Discovery App
A sleek, responsive web application that allows users to search for their favorite songs and instantly retrieve synchronized lyrics. Built with a focus on API Integration and Dynamic DOM Manipulation.

🚀 Project Overview
LyricFinder leverages a third-party music API to provide a seamless search experience. Users can search by artist or song title, view a list of top results with pagination, and fetch full lyrics with a single click—all without a page reload.

🛠️ Tech Stack
Front-End: Vanilla JavaScript (ES6+) / React (choose one)

Styling: CSS3 (Custom Flexbox/Grid) / Bootstrap

API: Lyrics.ovh or Musixmatch API

Tooling: Fetch API, Promises, and Async/Await

✨ Key Features
Live Search: Instant results using asynchronous data fetching.

Smart Pagination: Browse through multiple pages of search results efficiently.

Clean UI/UX: A minimalist, "Dark Mode" inspired design for better readability of lyrics.

Error Handling: Graceful "Not Found" states and loading spinners for a smooth user journey.

🏗️ Technical Challenges Solved
Asynchronous Flow: Mastered the use of async and await to handle multiple API endpoints (searching for songs vs. fetching specific lyrics).

CORS Management: Resolved cross-origin resource sharing issues to ensure stable data retrieval from third-party servers.
